robot_designer_plugin.export.urdf.urdf_export module¶

Operators (and functions) exporting to URDF. It supports the creation of ROS packages, zipped ROS packages and ‘plain’ URDF files with absolute or relative file paths.

Note: modify pyxb.binding.basis._TypeBinding_mixin.toxml() to call xml.dom.minidom.Node.toprettyxml()
