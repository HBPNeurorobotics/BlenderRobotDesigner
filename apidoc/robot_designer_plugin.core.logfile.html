robot_designer_plugin.core.logfile moduleÂ¶

This sub package provides logging mechanism for the plugin development.

Blender is not easy to debug (adding an external debugger is planned) such that providing informative output to a log file is mandatory. The log file is stored in the path stored in the plugin directory (config.script_path) in resources/log.txt.
