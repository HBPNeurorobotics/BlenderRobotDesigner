robot_designer_plugin.export.sdf.sdf_export module¶

Operators (and functions) exporting to SDF. It supports the creation of ROS packages, zipped ROS packages and ‘plain’ SDF files with absolute or relative file paths.

Note: modify pyxb.binding.basis._TypeBinding_mixin.toxml() to call xml.dom.minidom.Node.toprettyxml()
