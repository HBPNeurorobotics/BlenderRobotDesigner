robot_designer_plugin.core.config module¶

This submodule holds the configuration of the plugin (path variables, error message format, etc.)

robot_designer_plugin.core.config.resource_path = ‘/home/user/.config/blender/2.78/scripts/addons/robot_designer_plugin/resources’¶

This variable stores the file name where the plugin is defined (assuming it is the top level of the core.)
